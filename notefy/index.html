User Experience Visual Design Konzept, Idee und Recherche

Service

Folgende Teilbereiche wurden innerhalb des Projektes realisiert: Value Proposition Canvas, Interview, Benutzergruppen und Personas, User Journey Map, Research, User Story, Wireframe, User-Testing am LowFi- und HighFi-Prototype.

Deliverables

Durch festgelegten Schwerpunkt auf Human Centered Design kreierten wir im Team eine untersttzende Lernplattform fr Studierende und Schler*innen, welche ermglicht persnliche Notizen im Handumdrehen zu erstellen, strukturiert abzulegen, sowie kinderleicht wieder abzurufen und so das erneute Beschftigen mit dem so gespeicherten Wissen attraktiv und visuell ansprechend zu gestalten.

Der Benutzer steht im Zentrum

Die wichtigsten Kernfunktionen von Notefy sttzen sich auf den alltglichen Bedrfnissen von Studierenden und deren Pain Points, welche mittels Notefy abgefangen werden.

User Testing

Durch iteratives User Testing anhand eines LowFi- und HighFi-Prototypes wurde Notefy in einer Hochschule Hand in Hand mit einer mglichen Zielgruppe, und dessen Feedback angepasst und optimeirt, damit es die Bedrfnisse der Benutzer vollstndig abdeckt.
